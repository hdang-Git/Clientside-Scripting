Angular Triple Lab

  • Click here to see my Angular Triple Lab main page (Single Page Application).
  • Click here to invoke the List Web API - needs no input parameter, all person data will appear in browser through JSON Viewer plugin.
  • Click here to invoke the Delete Web API - expects parameter "id" with value specifying which record to delete. This API returns empty string if the record was sucessfully deleted. Otherwise, it shows the error message (possible database error, "0 records deleted", etc).
  • Click here to invoke the Insert Web API - expects parameter "jsonData" to hold a JSON-ized StringData object containing the data to be inserted. This API returns a JSON-ized StringData object full of error messages (or empty strings where there is no error). For example, if you provide a URL such as the one below, refresh the page, then see the following JSON data, it means that your insert worked (no error messages). 
             {
            personId: "",
            name: "",
            age: "",
            sex: "",
            errorMsg: ""
        }
  • Click here to invoke the Update Web API. Like insert, this API expects parameter "jsonData" to hold a JSON-ized version of StringData that describes the data to be updated. For update, a valid/existing personId must be supplied. Like insert, this API also returns a JSON-ized version of a StringData object, full of error messages (or empty string if no error). If there exists a person with id 109, the following URL should work to update a person record. 
        URL: tripleLab/webAPIs/personUpdate.jsp?jsonData={personId:'109',name:'Sallacious',age:34,sex:'Female'}
