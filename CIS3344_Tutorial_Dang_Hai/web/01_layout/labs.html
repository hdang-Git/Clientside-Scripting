CSS Techniques Utilized:

The advanced CSS techniques that were utilized for this application were rounded corners on the images and container. For the image, a box shadow was implemented with a gradient for blur effect. Also for body background, a gradient color was applied from top left to bottom right.

  • I also learned the magic of angular js which can be used to include code snippets into an html page.
